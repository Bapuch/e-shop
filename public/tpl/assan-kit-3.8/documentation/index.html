Php forms

We've using the smart forms into the assan bootstrap4 templates pack. which is provide the different type of forms like contact form with image captcha, contact form with google recaptcha, checkout, booking etc.

Basic contact form

you can see a form into pages/contact-basic.html how to make it working

  • Make sure your website is on server, php not working localy. so first you need to put the website on server for make contact form working
  • Make sure all the path you given into html/smart-form/contact/js/smart-form.js is correct check the paths below
  • Open html/smart-form/contact/php/settings.php and add your email to receiver email address - $receiver_email = "example@domain.com";
  • After making the changes - contact form will working fine as work on live demo
Advanced contact form with google recaptcha

Using reCAPTCHA users can prove they are human without solving a CAPTCHA. They need just a single click to confirm they are not a robot. So, reCAPTCHA will protect your website from spam with better user experience. You can easily integrate Google reCAPTCHA in PHP script.

Get reCAPTCHA API keys from https://www.google.com/recaptcha/admin

Register your site in image below

Get your Site Key - Site key is used to display the reCAPTCHA widget in HTML

Open contact-advanced.html from .html pages look for this below 

  1. <div id="g-recaptcha" class="g-recaptcha" data-sitekey="6LfNVScTAAAAACM_Brt8tzJBwh6A1nqCxqZLN0Px"></div>

Replace data-sitekey=" " with your key

Open php folder inside that form look for the settings folder look for this line below fill it with your secret key 

  1. 	$secretKey = "";

Open html/smart-form/contact-recaptcha/php/settings.php and add your email to receiver email address - $receiver_email = "example@domain.com";
